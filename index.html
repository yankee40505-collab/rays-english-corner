文法筆記

常考觀念，點開就懂

點開任一張卡片，看規則、看例句，搭配下面的練習加深印象。

必考
Both / Two / All
先數「總共幾個」，再決定用哪個詞
👤
必考
Each / Every one
一個一個算 → 動詞記得用單數
⚖️
可數 vs 不可數
news、money、homework 都是不可數！
🪣
a few / few / a little / little
有 a 是好消息，沒有 a 是壞消息
↔️
必考
Although / Though
雖然…但是，英文只能留一個！
⏪
不規則過去式
hurt → hurt、win → won 等常見變化
❓
附加問句
前正後負、前負後正，時態要一致
💬
that 名詞子句
I heard that… / I know that… 的用法
